A REST client written in node.js and express.

API Calls:


| Type | Route | Parameters |
| ---  | ---   | ---        |
| post | /writeTrainData | device_id, rssi, location |
| post | /writeData | device_id, rssi |
| post | /predict | search |

To run:

`npm install`

`bower install`

`node app.js`
